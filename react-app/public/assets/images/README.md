## Image Sources

You can find trail images from:
1. Unsplash (https://unsplash.com) - Free high-quality images
2. Pexels (https://www.pexels.com) - Free stock photos
3. National Park Service (https://www.nps.gov) - Public domain images
4. AllTrails - User-submitted trail photos (with proper attribution)

## Recommended Image Specs

- Format: JPG or PNG
- Dimensions: 800x600 pixels minimum (maintain 4:3 or 16:9 aspect ratio)
- File size: Under 500KB for optimal loading
- Quality: High resolution, landscape/nature photography
